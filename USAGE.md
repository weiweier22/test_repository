# Usage Guide / 使用指南

This document provides usage examples for the test repository.

本文档提供了测试仓库的使用示例。

## Python Script / Python 脚本

### Running the greeting script

```bash
python3 hello_world.py
```

### Using in your code

```python
from hello_world import greet, farewell

# Greet someone
message = greet("Alice")
print(message)  # Output: Hello, Alice! 你好，Alice！

# Say goodbye
farewell_msg = farewell("Bob")
print(farewell_msg)  # Output: Goodbye, Bob! 再见，Bob！
```

## Features / 功能

- ✅ Simple greeting functionality
- ✅ Bilingual support (English/Chinese)
- ✅ Easy to extend and customize
